# Rahul International School Website

A modern landing page for Rahul International School, built with React, Tailwind CSS, GSAP, and Vite.

This project is designed to showcase the school's mission, campus experience, and enrollment call-to-action for parents and families.

## Overview

- Responsive school website layout
- Animated hero section with video and headline
- Branded school messaging for parents and students
- Smooth scroll-triggered transitions and image/video presentation

## Tech Stack

- React
- Tailwind CSS
- GSAP
- Vite

## Features

- Hero section tailored for Rahul International School
- Animated text and visual motion for attention
- Embedded campus video preview
- Responsive design for desktop and mobile
- Simple call-to-action button for enrollment interest

## Project Structure

- `src/App.jsx` – application entry
- `src/main.jsx` – React render setup
- `src/components/` – hero, about, features, contact, footer and related UI components
- `src/index.css` – project styling and utility classes
- `public/` – static images, fonts, and video assets

## Quick Start

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal to preview the school website.

## Notes

- Use `/img/about-ris.png` for the About section image.
- Update the hero and about text to reflect Rahul International School content.
- Remove any placeholder branding or unrelated tutorial references.
